# Backend Structure

## Directories
- **config.js**: Configuration settings for the backend.
- **middleware/**: Contains middleware functions for request validation and processing.
- **models/**: Database models and queries.
- **routes/**: API route handlers.
- **services/**: Business logic and service layer.
- **utils/**: Utility functions shared across the backend.
- **server.js**: Entry point for the backend server.
- **socket.js**: Real-time communication logic using Socket.IO.
